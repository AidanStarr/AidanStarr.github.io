Miscellaneous Code

  • .m
    • MISPLOT.m: Short function to plot Marine Isotope Stages as shaded bars on the current axis. Uses MIS boundaries from LR04
    • OXYPLOT.m: Quick script to plot and format d18O time series
    • D13SLICE.m: Generate a 3-D visualisation of ocean d13C of DIC data for a specified region. Uses the d13C climatology of Eide et al., 2017
    • redfitAS.m: A script (function coming soon) to replicate Schulz and Mudelsee's REDFIT algorithm. Lomb-Scargle Periodogram of irregularly-spaced time series with red noise error estimates and bias-correction based on Schulz and Mudelsee 2002
  • .tex
    • template.tex: The template for my latex-compiled notes, in the tufte-class. This format (with a wide margin and numerous 'sidenotes') is inspired by Edward Tufte, a deity of information design. Modified from Tufte-latex
